# IntervalCoach

準備・エクササイズ・休憩の時間とラウンド数を設定するだけで、開始／終了のカウントダウンを音声で案内し、フェーズに合わせたBGMを自動生成して流すインターバルトレーニングPWA。UIは英語、音声ガイドは日本語（セリフは`SPEECH`辞書に集約済みで多言語化しやすい構造）。

- 準備・エクササイズ・休憩の時間（既定60秒）とラウンド数を指定。ラウンド数は∞（無限）にもでき、その場合はStopで手動終了
- フェーズ開始を音声でアナウンスし、ラスト3秒を「3・2・1」とカウントダウン
- BGMはWeb Audio APIでリアルタイム生成（音源ファイル不要）。エクササイズ中はマイナー調118BPMのクールなグルーヴ、休憩中はパッド＋チャイムの癒し系アンビエント
- 音声再生中はBGMを自動でダッキング（音量を一時的に下げる）
- タイマー画面はビートに同期した波紋アニメーション付き（BGMオフ時は秒針に同期）
- ワークアウト履歴を自動記録（連続日数・累計・週間回数）。設定画面に進捗カードと励ましメッセージを表示
- 完走時は実績に応じた褒めメッセージ（初回・マイルストーン・連続記録など）＋花火アニメーション＋拍手・ファンファーレ効果音
- ∞モードでは1ラウンド以上完了後のStopを「完了」として記録・お祝い
- 画面スリープ防止（Wake Lock）、設定の自動保存、オフライン動作（Service Worker）

## iPhoneでの音声について

iOSではSafari・Chrome・Firefoxなど全てのブラウザがApple提供のWebKitエンジンを使う仕様のため、読み上げ（Web Speech API）の実体もOS標準の音声合成エンジンに委ねられます。ブラウザの種類による違いは基本的にありません。ただし画面ロックやバックグラウンド遷移で音声・BGMが停止する場合があるため、運動中は画面をつけたまま使うことを想定しています（アプリ側でWake Lockを要求してスリープを防止）。

## 使い方

ブラウザで`index.html`を開くだけ。スマホなら「ホーム画面に追加」でアプリとしてインストールできます。

依存ライブラリなし・ビルド不要の単一HTMLアプリです。
